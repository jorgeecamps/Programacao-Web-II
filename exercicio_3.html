Qual é a diferença entre linguagem de programação Server Side e Client Side?

Client Side

Com relação à linguagem do lado Cliente, temos o JavaScript para que seja possível manipular a página do usuário diretamente, podemos emitir uma mensagem na tela, validação de um formulário, um exemplo de validação de formulário são apresentar mensagens ao usuário quando os campos obrigatórios não são preenchidos.

Server Side

Com relação à linguagem do Servidor, temos o PHP, quando precisamos interagir com um banco de dados. exemplo, quando você acessa um site que publica notícias, muito provavelmente, este site armazena as notícias em uma base de dados.

Conclusão

A grande diferença está onde ocorre o processamento do código, com evolução da web, cada vez mais o cliente está presente e interagindo com o aplicacão, ambas as linguagens estão presente em grande maioria das aplicações, essas linguagens ajudam na dinâmica da aplicacão e a torna mais única e atrativa, não sobrecarrega o servidor e etc.
